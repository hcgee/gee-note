# 居中对齐

> position 结合 transform

```css
position: absolute;
left: 50%;
top: 50%;
transform: translate(-50%, -50%);
```

> flex 布局

```css
display: flex;
justify-content: center;
align-items: center;
```

<!-- > grid 布局

```css
display: grid;
text-self: center;
justify-self: center;
``` -->

> 水平居中

```css
/* 行内元素 */
text-align: center;
```

```css
margin: 0 auto;
```

> 垂直居中

```css
/* 单行文本 */
height: 20px;
line-height: 20px;
```
